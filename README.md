# Как начать свой YouTube-канал — интерактивный сервис

Не просто курс, а четыре режима в одном файле:

- **Тренажёр** — 12 уроков в 4 юнитах (Фундамент → Запуск → Рост → Масштабирование), 36 действий с примерами «как сделал бы профи» и «как не стоит». Урок открывается только после того, как пройден предыдущий. Прогресс сохраняется в браузере.
- **Игра** — собираешь связку ниша + ЦА + стратегия (144 комбинации), получаешь рейтинг (лигу и процентиль) и разбор по 4 параметрам: охват, скорость роста, монетизация, риск.
- **Мастерская** — собираешь промт для сценария видео из приёма хука (бисоциация, факт, вопрос, парадокс) и структуры (простая или круг Дэна Хармона).
- **Документы** — чек-лист оформления канала, шаблон сценария, 50 идей для видео, бриф для фрилансера. Каждый копируется одной кнопкой.

Плюс промт для ИИ в конце каждого юнита тренажёра.

## Как посмотреть

Открой `index.html` в браузере — сборка и зависимости не нужны.

Для публичной ссылки — включи GitHub Pages в настройках репозитория (см. ниже).

## Структура

- `index.html` — весь сервис: разметка, стили и логика в одном файле.

## Технологии

Чистый HTML/CSS/JS, без библиотек и фреймворков. Прогресс сохраняется через localStorage.
